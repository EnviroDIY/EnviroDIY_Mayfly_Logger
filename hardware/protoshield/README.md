EagleCAD files for the Mayfly Protoshield v1 (discontinued in 2020)

NOTE:  This ProtoShield is only compatible with the older Mayfly v0.5b boards.  It is not compatible with the newer Mayfly v1.0 or v1.1 board due to changes in the pinouts of the headers.  Do not attempt to use this ProtoShield with a Mayfly v1.0 or v1.1 board or damage to the Mayfly and/or ProtoShield will occur.
